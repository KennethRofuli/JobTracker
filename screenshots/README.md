# Screenshots

This folder contains screenshots for the Job Tracker README.

## Required Screenshots:

1. **dashboard.png** - Main dashboard view showing the application table
2. **stats-cards.png** - Statistics cards at the top of dashboard
3. **extension-popup.png** - Browser extension popup interface
4. **login-page.png** - Google OAuth login page
5. **search-filter.png** - Search and filter controls in action
6. **auto-capture.png** - Extension auto-capturing a job from Indeed/LinkedIn

## How to Add Screenshots:

1. Take screenshots of your application
2. Save them with the names listed above
3. Place them in this folder
4. They will automatically appear in the README

## Screenshot Guidelines:

- Use PNG format for clarity
- Recommended size: 1200px-1600px wide
- Show realistic data (not empty states)
- Blur any personal information if needed
